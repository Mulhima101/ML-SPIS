# ML-Based Student Progress Improvement System - Backend

This is the Flask backend for the ML-Based Student Progress Improvement System. It provides the API for the React frontend to interact with.

## Setup Instructions

1. Create a virtual environment:
python -m venv venv

2. Activate the virtual environment:
venv\Scripts\activate

3. Install dependencies:
pip install flask flask-sqlalchemy flask-cors werkzeug pandas scikit-learn numpy

4. Run the application:
python run.py
